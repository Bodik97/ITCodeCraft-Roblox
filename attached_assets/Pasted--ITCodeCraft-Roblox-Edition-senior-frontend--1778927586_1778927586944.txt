🎯 ЗАВДАННЯ: Створити односторінковий лендінг для ITCodeCraft Roblox Edition

Ти — senior frontend-розробник. Створи production-ready односторінковий лендінг для онлайн ІТ-школи **ITCodeCraft**, яка навчає дітей 8–16 років програмуванню ігор у **Roblox Studio** мовою **Lua**.

Головна ціль сайту — **конверсія в заявку на безкоштовний пробний урок**.

## 📦 ТЕХНІЧНИЙ СТЕК (обов'язково)

- **Astro 5.x** (статичний генератор, SSG режим)
- **TypeScript** (strict mode)
- **Vanilla CSS** з CSS-змінними (без Tailwind — Astro дає кращу продуктивність)
- **Мінімум JavaScript** — тільки там, де треба інтерактив (форма, акордеон, мобільне меню)
- **`@astrojs/sitemap`** — для SEO
- **`@astrojs/partytown`** — для аналітики у Web Worker (без блокування main thread)
- **`astro:assets`** — для оптимізації зображень (WebP/AVIF auto-conversion)

### Цільові метрики (acceptance criteria)
- **Lighthouse Performance:** 95+
- **Lighthouse SEO:** 100
- **Lighthouse Accessibility:** 95+
- **First Contentful Paint:** < 1.0s
- **Largest Contentful Paint:** < 1.8s
- **Total Blocking Time:** < 100ms
- **JavaScript bundle:** < 30KB (gzipped)

## 🎨 ДИЗАЙН-СИСТЕМА

### Концепція
**Двошарова естетика:**
- Для дітей: яскраво, грайливо, ігрові елементи (бейджі, "ачівки", блоки)
- Для батьків: структуровано, з цифрами, відгуками, гарантіями

### Кольорова палітра (CSS variables)
```css
--color-primary: #6366F1;       /* Indigo */
--color-primary-dark: #4F46E5;
--color-accent: #FBBF24;         /* Amber для CTA */
--color-roblox: #00A2FF;         /* Roblox-стилістика */
--color-success: #00B06F;        /* PUBLISHED статус */
--color-bg: #FFFFFF;
--color-bg-soft: #F9FAFB;
--color-bg-dark: #0F172A;        /* Hero та секція ігор */
--color-text: #1F2937;
--color-text-soft: #6B7280;
```

### Шрифти (Google Fonts)
- **Fredoka** (400, 600, 700) — заголовки (грайливий, Roblox-вайб)
- **Inter** (400, 500, 600) — body
- **JetBrains Mono** — для коду, цифр статистики

### Візуальні елементи
- **Блокові 3D-форми** — імітація Roblox-стилю через CSS transforms
- **М'які тіні** з offset (як 3D-кнопки в іграх)
- **Бейджі-ачівки** з glow-ефектом
- **Скриншоти** Roblox Studio інтерфейсу
- **Аватари дітей у Roblox-стилі** (квадратні голови, блокові тіла)

### Принципи стилю
- НЕ використовуй gradient'и фону (тільки solid кольори)
- Тіні мають бути **offset shadows** (3D-кнопковий ефект): `box-shadow: 0 4px 0 #B8830C`
- Округлення: 12–16px для карток, 9999px для бейджів
- Анімації: тільки на hover і fade-in при скролі (Intersection Observer)

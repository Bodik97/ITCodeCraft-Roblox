# @workspace/itcodecraft

Astro-лендінг. Детальна карта репозиторію — у [кореневому README](../../README.md).

## Локально

```bash
pnpm run dev      # http://localhost:5173
pnpm run check
pnpm run build    # → dist/
```

## Де що лежить

| Папка | Призначення |
|-------|-------------|
| `src/pages/index.astro` | Збірка сторінки |
| `src/sections/` | Секції (Hero, FAQ, …) |
| `src/data/data.json` | Контент |
| `src/components/` | Layout, Logo, форма, модалки |
| `src/styles/` | Глобальні стилі |
| `src/assets/` | PNG-зображення |
| `public/` | favicon, robots, OG |
